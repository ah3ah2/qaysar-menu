# Qaysar Pizza Menu - VS Code Workspace

## Run
1) Open this folder in VS Code
2) Install/enable Live Server
3) Right click `index.html` -> Open with Live Server

## Notes
- Put your images inside `images/` with the exact same names used in `app.js`.
- The 3 new tabs are:
  - الاصناف الجانبيه (sides)
  - اخر العروض (offers)
  - الاصناف الجديده (new)
